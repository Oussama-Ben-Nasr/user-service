# This project is a simple implemenation of user registration endpoints, using fast-api.
# routes
1. login
    An enpoint taking the login information
        + username
        + password
    returns
        -> valid registred user
        -> invalid credintials

2. register
    An enpoint taking the login information
        + username
        + password
    returns
        -> registers a user with the given credentials.
